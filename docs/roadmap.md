# Roadmap

## v0.1 (UI foundation)
- Clean multi-tool layout (Interceptor / Repeater / Extender)
- Request builder + response viewer
- LocalStorage persistence for repeater list

## v0.2 (quality)
- Header editor improvements (add/remove rows, import/export)
- Response rendering: JSON pretty print, content-type handling
- Diff view for repeater responses

## v0.3 (backend companion - optional)
- Local proxy endpoint to bypass CORS and enable richer HTTP
- Cookie jar, redirect tracing, timing metrics

## v0.4 (extender)
- Plugin registry UI
- Signed/verified extensions (later)
